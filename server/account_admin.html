第一版先做只读查询，避免后台误操作；service token 仍从服务端环境变量配置。

Service Token

权限范围（可选）：不确定就先不选，后续可以用同一个服务标识重新生成。
token 明文只在创建或轮换时展示一次。重新使用同一个服务标识会轮换旧 token。

访问方式

在本机建立 SSH 隧道后访问下面的本地地址：

ssh -L 26061:127.0.0.1:26061 <server-alias> http://127.0.0.1:26061/local/account-admin
