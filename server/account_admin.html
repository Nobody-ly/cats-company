按 UID 精确查询

精确查询用于快速确认单个账号信息和状态；资料字段仍保持只读，避免误操作。

Service Token

Service Token 是内部服务访问账号中心的凭证，不属于某个用户；不确定权限时可以先不选。同一服务标识再次提交会轮换 token，并让旧 token 失效。
token 明文只在创建或轮换时展示一次。它应该只放在业务服务端 secret 或环境变量里，不能下发给浏览器。

运营边界

这块先做商业化账本、灰度入口和 relay 对账。只有开启 enforce 变量后，“尝试同步”才会把模型额度写到 relay-admin 的 provider config；默认不会替换现有用户的 relay key、模型调用、key 级月预算或 Bifrost 路由。通用账本额度暂时只做记录和展示，不作为 relay 强制限额。已超额用户需要补额或在 relay 侧重置用量后再继续。

访问方式

在本机建立 SSH 隧道后访问下面的本地地址：

ssh -L 26061:127.0.0.1:26061 <server-alias> http://127.0.0.1:26061/local/account-admin
